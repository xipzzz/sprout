Never color alone — correct / wrong

Color is a reinforcement, never the only signal. Every feedback state carries an icon + label + position, so it reads for the ~8% with color-vision deficiency and anyone in bright sun. The right column is the same UI in grayscale — still unambiguous.

Standards & focus order
Contrast
Body + secondary text meet WCAG AA (4.5:1 text, 3:1 large/UI). Audit soft button fills + muted text on cream.
Color independence
Correct/wrong = icon + label + position, never hue alone.
Dynamic Type
Reflow to system size by default; cards grow vertically; test to XXXL — no truncation.
Touch targets
≥ 44×44pt hit area on every control, even small glyphs.
Focus order
Exercises read prompt → options → Check; sheets trap focus and return it on close.
Reduce Motion
Honored from the Motion doc — particles/springs become a fade; haptics still fire.
