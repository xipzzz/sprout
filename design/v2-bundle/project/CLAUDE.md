# Sprout Learning Hub — Project Memory

## Language
- **English only for all copy and content** — no Spanish, French, or other languages in examples, explanations, or UI text.
- Exercises should teach/reinforce English (vocabulary, grammar, sentence structure).
- Do not use Spanish words like "el gato", "hola", etc. in feedback or prompts.

## Brand
- App is **Sprout** (not Duolingo). Mascot is **Pip**, a friendly green sprout.
- Colors live in `SPROUT` token object in `shared.jsx`.
- Cream background `#F8F1E0`, green `#58CC42`, sun `#F5C23D`, coral `#F57C7C`, sky `#A8D8EA`.

## Tone
- Warm, encouraging, slightly playful. Never punishing or alarming.
- "Calm mode" tweak replaces streak/timers with gentle growth framing.
