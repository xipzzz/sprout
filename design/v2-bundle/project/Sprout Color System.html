Rules
One source
Every color routes through a token (SPROUT.*) — no per-screen hard-coded hexes.
Brand vs feedback
Chrome/brand green ≠ correct-green. Correct is deeper so feedback never reads as decoration.
Never color alone
Correct/wrong always pair color with a ✓/✗ icon + label (color-blind-safe).
Small accent set
Water / gems / gold / rare only — a deliberate few so the playful colors don’t clash.
Dark later
Light-only now; the Moonlit set is a parallel token map — re-enabling dark is just swapping tokens.
