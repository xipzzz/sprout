Sprout · Button System

Buttons

One shape, three tiers, clear states. Filled green = go, text = gentle, red = careful. Importance is always obvious at a glance.

The three tiers

Primary

Filled green pill

The one main action on a screen — Continue, Check, Start. Full-width green pill with a solid drop shadow that presses down on tap. Only one primary per screen.

Enabled
Disabled

Secondary

Tonal / outline

An alternate action that sits beside the primary — “Review”, “Set up later”, “Keep my garden”. Same pill shape, outlined on paper so it reads as the quieter choice.

Tertiary

Text only

The gentlest action — declines and skips in Pip’s calm voice: “No thanks”, “Skip”, “Can’t listen now”. Text only, no container, so it never competes with the primary.

Rules
One shape
Every button: 56px tall, 16px radius, full-width pill (tertiary/text is 48px, no container). No stray sizes.
One primary
Exactly one filled-green primary per screen — the single most important next step.
Disabled → enabled
Primary fades (desaturated green) until valid, then snaps to full green. Check is disabled until an answer is chosen.
Press feedback
Every button presses down (shadow collapses ~3px) on tap; pairs with a light haptic.
Casing & weight
Primary/secondary labels uppercase + extrabold; tertiary sentence-case. 1–3 words.
Destructive
Red text/outline only; filled red reserved for a final “Delete forever” confirm.
